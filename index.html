Task Management System

Instructions for using the Task Manager

  • Add new task:

    You can create tasks with a task name, priority level (Low, Medium or High) and the name of the staff member assigned to it.

  • Display tasks:

    Tasks are listed, including name, priority level and staff member. Each task includes controls to complete, edit, or delete it:

    • Complete task:

      Click the checkbox to mark or unmark the task as completed. The task will apear in the list of completed tasks while checked.

    • Delete task:

      Click the trash can to remove tasks no longer needed.

  • Search tasks:

    You can look up tasks by typing any task attribute to display results.
