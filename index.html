Task Management System

Instructions for using the Task Manager

  • Add new task:

    You can create tasks (via a form) with a task name, due date, priority level (Low, Medium or High) and optional consultant (enter the exact role from a list).

  • Display tasks:

    Tasks are listed, including name, due date, priority level and consultant. Each task includes controls to either complete or delete it:

    • Complete task:

      Select the control to mark tasks as complete. The task will remain visible with strikethrough formatting.

    • Delete task:

      Select the control to remove tasks no longer needed.

  • Search tasks:

    You can look up tasks by typing the exact task name to display results or the message ‘Task not found’.
